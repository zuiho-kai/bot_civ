## 用户个人资料卡片 UI 设计稿

### 1. 组件结构

```
UserProfileCard/ (主容器，表示单个用户的个人资料卡片)
  ├── UserAvatar (用户头像，包含图像和状态指示)
  ├── UserInfo/ (用户基本信息区域)
  │   ├── UserName (用户显示名称)
  │   └── UserCredits (用户信用点余额显示)
```

### 2. 布局规格

**整体布局思路:** 卡片内部元素垂直堆叠，并水平居中。

*   **`UserProfileCard`**
    *   **尺寸:** 建议宽度 `240px` (与 `--info-panel-bg` 的默认宽度保持一致)。高度 `auto`，根据内容自适应。
    *   **布局:** `display: flex; flex-direction: column; align-items: center; justify-content: flex-start;`
    *   **内边距 (Padding):** `padding: 16px;` (上下左右)
    *   **间距 (Gap):** `gap: 16px;` (卡片内各主要元素之间的垂直间距)

*   **`UserAvatar`**
    *   **尺寸:** `width: 64px; height: 64px;` (固定尺寸)
    *   **布局:** 图像居中。

*   **`UserInfo`**
    *   **尺寸:** `width: 100%;` (填充父容器宽度)
    *   **布局:** `display: flex; flex-direction: column; align-items: center;` (名称和信用点垂直堆叠并居中)
    *   **间距 (Gap):** `gap: 8px;` (名称与信用点之间的垂直间距)

*   **`UserName`**
    *   **尺寸:** `auto`
    *   **布局:** `text-align: center;`

*   **`UserCredits`**
    *   **尺寸:** `auto`
    *   **布局:** `display: inline-flex; align-items: center; justify-content: center;` (作为徽章，内容水平居中)
    *   **内边距 (Padding):** `padding: 4px 8px;` (徽章内部的间距)

### 3. 视觉规格

*   **`UserProfileCard`**
    *   **背景色:** `var(--card-bg)`
    *   **边框:** `1px solid var(--border-subtle)`
    *   **圆角:** `var(--radius-md)` (8px)
    *   **阴影:** `var(--shadow-sm)`

*   **`UserAvatar`**
    *   **圆角:** `50%` (使头像呈圆形)
    *   **边框:** `2px solid var(--status-online);` (示例为在线状态，颜色会根据实际用户状态变化：`--status-online`, `--status-busy`, `--status-offline`)

*   **`UserName`**
    *   **文字色:** `var(--text-primary)`
    *   **字体大小:** `16px`
    *   **字重:** `font-weight: bold;`

*   **`UserCredits`**
    *   **背景色:** `var(--credits-bg)` (浅色背景作为徽章底色)
    *   **文字色:** `var(--credits-text)` (深色文字)
    *   **圆角:** `var(--radius-sm)` (4px)
    *   **字体大小:** `12px`
    *   **字重:** `font-weight: 500;` (中等粗细)

### 4. 交互状态

*   **`UserProfileCard` (假设其可以点击或作为列表项)**
    *   **`hover` 效果:**
        ```css
        .UserProfileCard:hover {
          background-color: var(--bg-secondary); /* 稍微变深 */
          box-shadow: var(--shadow-md); /* 阴影增强 */
          transform: translateY(-2px); /* 轻微上浮 */
          transition: all 0.2s ease-in-out; /* 平滑过渡 */
        }
        ```
    *   **`loading` 状态:**
        当数据加载时，可以在 `UserInfo` 区域显示骨架屏。
        ```html
        <!-- HTML 结构示例 -->
        <div class="UserProfileCard">
          <div class="UserAvatar --loading"></div>
          <div class="UserInfo">
            <div class="UserName --loading"></div>
            <div class="UserCredits --loading"></div>
          </div>
        </div>
        ```
        ```css
        .--loading {
          background-color: var(--bg-secondary); /* 骨架屏底色 */
          border-radius: var(--radius-sm);
          /* 添加动画效果 */
          animation: skeleton-loading 1.5s infinite linear alternate;
        }
        .UserAvatar.--loading {
          border-radius: 50%; /* 保持圆形 */
        }
        .UserName.--loading {
          width: 80%; /* 模拟文本长度 */
          height: 16px; /* 模拟字体高度 */
        }
        .UserCredits.--loading {
          width: 60px; /* 模拟徽章长度 */
          height: 20px; /* 模拟徽章高度 */
        }
        @keyframes skeleton-loading {
          0% { opacity: 0.7; }
          100% { opacity: 1; }
        }
        ```

### 5. 关键 CSS 片段

```css
/* UserProfileCard - 用户个人资料卡片容器 */
.UserProfileCard {
  display: flex;
  flex-direction: column;
  align-items: center;
  padding: 16px;
  gap: 16px; /* 头像与信息区之间的间距 */
  background-color: var(--card-bg);
  border: 1px solid var(--border-subtle);
  border-radius: var(--radius-md);
  box-shadow: var(--shadow-sm);
  width: 240px; /* 假设与右侧信息栏宽度匹配 */

.UserAvatar {
  width: 64px;
  height: 64px;
  border-radius: 50%;

  background-position: center;

/* UserInfo - 用户信息容器 (名称和信用点) */
.UserInfo {
  display: flex;
  flex-direction: column;
  align-items: center;
  gap: 8px; /* 名称与信用点之间的间距 */
  width: 100%;
}

/* UserName - 用户名称 */
.UserName {
  color: var(--text-primary);
  font-size: 16px;
  font-weight: bold;
  text-align: center;
  white-space: nowrap; /* 防止名称换行 */

/* UserCredits - 信用点余额徽章 */
.UserCredits {
  display: inline-flex;
  align-items: center;
  justify-content: center;
  padding: 4px 8px;
  background-color: var(--credits-bg);
  color: var(--credits-text);
  border-radius: var(--radius-sm);
  font-size: 12px;
  font-weight: 500;
  white-space: nowrap;
}
```

### 6. 验收标准 (供 Gemini 美学审查用)

*   [ ] 整个卡片背景色应为 `var(--card-bg)`，并带有 `var(--shadow-sm)` 的阴影效果。
*   [ ] 用户头像必须是正圆形 (`border-radius: 50%;`)，且带有 `2px` 宽度的边框，其颜色应能清晰反映用户状态（如在线为 `var(--status-online)`）。
*   [ ] 用户名称字体大小为 `16px` 粗体 (`font-weight: bold;`)，颜色为 `var(--text-primary)`，并确保文本水平居中。
*   [ ] 信用点余额应显示为一个带有 `var(--credits-bg)` 背景、`var(--credits-text)` 文字颜色、以及 `var(--radius-sm)` 圆角的徽章样式。
*   [ ] 卡片内部的头像与用户信息区域之间应有 `16px` 的垂直间距，用户信息区域内的名称与信用点之间应有 `8px` 的垂直间距，所有内容均垂直居中对齐。
